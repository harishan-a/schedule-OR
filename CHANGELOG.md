# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] — 2025-04-01

### Added

- **Schedule Management** — Day, week, month, and TV display calendar views with Syncfusion Flutter Calendar
- **Multi-Role Authentication** — Firebase Auth with Doctor, Nurse, Technologist, and Admin roles
- **Real-Time Notifications** — Push notifications (FCM), SMS (Twilio), and email (SendGrid)
- **Resource Conflict Detection** — Automatic room, staff, and equipment overlap checks before scheduling
- **Staff Directory** — Searchable, filterable directory of all medical personnel
- **Equipment Catalog** — Track surgical equipment availability and assignments
- **Patient Lookup** — Search across patient records
- **Surgery Log & Audit Trail** — Complete history of surgical operations with change tracking
- **Dark Mode & Accessibility** — System-aware theming with accessible color schemes
- **CSV Bulk Import** — Import schedules and staff data from CSV files
- **Local Development Environment** — Firebase emulator setup with Makefile automation and seed data
